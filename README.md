# PSP_PracticaPrimerTrimestre_2

## Aplicación 2

### Ejecución
Abrir `cmd` o `PowerShell` y ejecutar el siguiente comando con la ruta de su proyecto: `java -jar C:\Path\PSP_PracticaPrimerTrimestre_2\PSP_App-2.0.jar`

#### Inicio
Se iniciará un menú en el que el usuario deberá introducir un número según la opción que desee.

#### Opción 1:
Lee de manera secuencial todos los registros y muestra el tiempo empleado en la lectura y la suma (sin usar sentencias SQL de suma) del
campo INGRESOS. 

#### Opción 2:
Lee la base de datos con 5 hilos de ejecución concurrentes y muestra el tiempo empleado en la lectura y la suma (sin usar
sentencias SQL de suma) del campo INGRESOS.

#### Opción 0:
Salir de la aplicación
